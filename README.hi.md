# MDeX v1.3.0 (macOS · Windows · Linux · पूर्णतः ऑफ़लाइन · Tauri v2)

> **MDeX** · उच्चारण "em-dex" (/ˌemˈdɛks/) — अक्षर M के बाद "dex", दो अक्षर।

**एयर-गैप्ड / इंट्रानेट / डिस्कनेक्टेड** उपयोग के लिए एक ऑफ़लाइन-फ़र्स्ट Markdown रीडर और एडिटर। हर फ़ाइल स्थानीय रूप से प्रोसेस होती है — **कोई नेटवर्क नहीं, कोई अपलोड नहीं, कोई क्लाउड सिंक नहीं**।

- एक सिंगल सेल्फ़-कंटेन्ड HTML फ्रंटएंड (Vue / React नहीं); Tauri v2 केवल नेटिव शेल (विंडोज़, मेन्यू, फ़ाइल डायलॉग) प्रदान करता है।
- **ज़ीरो रनटाइम नेटवर्क रिक्वेस्ट**: `marked` / `KaTeX` / `highlight.js` / `DOMPurify` / `mermaid` / `jsPDF` / `html2canvas-pro` / `turndown` / `@retorquere/bibtex-parser` तथा सभी KaTeX woff2 फ़ॉन्ट एक ही `index.html` में इनलाइन / base64-एम्बेड किए गए हैं।
- `.md` / `.markdown` / `.html` समर्थित; इसे डिफ़ॉल्ट `.md` हैंडलर के रूप में सेट किया जा सकता है — खोलने के लिए डबल-क्लिक करें।

> उपयुक्त: **अनुसंधान / इंट्रानेट / गोपनीयता सुरक्षा** परिदृश्यों के लिए। कोई विज्ञापन नहीं, कोई टेलीमेट्री नहीं, कोई डेटा अपलोड नहीं।

## 📝 चीट शीट

**Markdown**: शीर्षक `# / ## / ###`, बोल्ड `**text**`, इटैलिक `*text*`, स्ट्राइकथ्रू `~~text~~`, इनलाइन कोड `` `code` ``, कोड ब्लॉक (ट्रिपल बैकटिक, वैकल्पिक भाषा के साथ), उद्धरण `> text`, सूचियाँ `- / 1.`, टास्क सूची `- [ ] / - [x]`, लिंक `[text](url)`, छवि `![alt](url)`, डिवाइडर `---`, टेबल `| A | B |`।

**गणित**: इनलाइन `$E = mc^2$`; ब्लॉक `$$\int_0^1 x\,dx$$` (कई पंक्तियों में फैल सकता है)। LaTeX सिंटैक्स का उपयोग, KaTeX द्वारा रेंडर; कोड ब्लॉक के अंदर `$` गणित डिलीमीटर के रूप में नहीं माना जाता। `align` / `aligned`, मैट्रिसेस, `cases` तथा अन्य सामान्य एनवायरनमेंट को समर्थन।

**उद्धरण**: टेक्स्ट में `[@key]` या `[@a; @b]` लिखें (LaTeX-संगत `\cite{key}`), लाइब्रेरी को एक ` ```bibtex ` ब्लॉक के माध्यम से एम्बेड करें या "Refs" बटन से कोई `.bib` लोड करें। अंत में एक संदर्भ सूची जनरेट की जाती है; इन-टेक्स्ट `[n]` क्लिक करने योग्य है।

## 📤 एक्सपोर्ट (इस रूप में सहेजें)

"Save As" पर क्लिक करें और कोई फ़ॉर्मैट चुनें:

- **Markdown (.md)**: स्रोत सहेजें और वर्तमान टैब का नाम / पथ अपडेट करें।
- **HTML (.html)**: इनलाइन CSS + कोड हाइलाइटिंग के साथ सेल्फ़-कंटेन्ड HTML; गणित `$…$` लिटरल के रूप में रखा गया, इनलाइन KaTeX द्वारा ऑटो-रेंडर।
- **PDF वेक्टर**: सिस्टम प्रिंट डायलॉग, वेक्टर आउटपुट, किसी भी ज़ूम पर क्रिस्प। "Save as PDF" चुनें।
- **PDF रास्टर**: सीधे एक PDF फ़ाइल जनरेट करता है (कोई डायलॉग नहीं), लो / मीडियम / हाई रिज़ॉल्यूशन के साथ; ब्लॉक सीमाओं पर पेजिनेट (कोई टूटा हुआ समीकरण / शीर्षक / कोड नहीं)।
- **LaTeX (.tex)**: कम्पाइल करने योग्य `.tex` स्रोत में कन्वर्ट (documentclass और पैकेज के साथ; गणित जैसा-है-वैसा रखा गया)। एक कॉपी एक्सपोर्ट करता है।

### macOS पर अनसाइन्ड ऐप खोलना (Gatekeeper बायपास)

यह ऐप डेवलपर-साइन्ड / नोटराइज़्ड **नहीं** है (ऑफ़लाइन परिदृश्य आमतौर पर ऑनलाइन नोटराइज़ नहीं कर सकते)। macOS 12+ पर, **विशेष रूप से macOS 26 (Tahoe)** पर, इसे सीधे `.dmg` से — या एक नई कॉपी किए गए बिल्ड से — लॉन्च करना **"MDeX.app is damaged and can't be opened."** के साथ विफल हो जाता है। वह Gatekeeper है, वास्तविक क्षति नहीं। Terminal में ठीक करें:

1. **पहले `MDeX.app` को `.dmg` से बाहर `/Applications` में खींचें** — इसे dmg से सीधे कभी न चलाएं (यह App Translocation और `com.apple.provenance` एट्रिब्यूट को ट्रिगर करता है, macOS 26 पर "damaged" का वास्तविक कारण)।
2. एट्रिब्यूट साफ़ करें और पुनः साइन करें:
   ```bash
   xattr -cr /Applications/MDeX.app
   codesign --force --deep --sign - /Applications/MDeX.app
   ```
   > `com.apple.provenance` SIP-संरक्षित है और `sudo` से भी **हटाया नहीं** जा सकता; पुनः साइनिंग सिग्नेचर रीसेट करती है इसलिए Gatekeeper इसे चलने देता है। `spctl` तब भी एड-हॉक साइनिंग के लिए `rejected` रिपोर्ट करता है — अपेक्षित, और यह `open` को **ब्लॉक नहीं** करता।
3. `open /Applications/MDeX.app` से लॉन्च करें (या डबल-क्लिक)। पहली बार लॉन्च पर अभी भी एक बार प्रॉम्प्ट हो सकता है — **System Settings → Privacy & Security → Open Anyway** से पुष्टि करें, या ऐप पर राइट-क्लिक → **Open**।
