# MDeX v1.3.0 (macOS · Windows · Linux · ਪੂਰੀ ਤਰ੍ਹਾਂ ਆਫਲਾਈਨ · Tauri v2)

> **MDeX** · ਉਚਾਰਨ "em-dex" (/ˌemˈdɛks/) — ਅੱਖਰ M ਤੋਂ ਬਾਅਦ "dex", ਦੋ ਉਚਾਰਾਂ।

**ਏਅਰ-ਗੈਪਡ / ਇੰਟ੍ਰਾਨੈੱਟ / ਡਿਸਕਨੈਕਟਡ** ਵਰਤੋਂ ਲਈ ਇੱਕ ਆਫਲਾਈਨ-ਫਸਟ Markdown ਰੀਡਰ ਤੇ ਐਡੀਟਰ। ਹਰ ਫਾਈਲ ਲੋਕਲ ਤੌਰ 'ਤੇ ਪ੍ਰੋਸੈਸ ਹੁੰਦੀ ਹੈ — **ਕੋਈ ਨੈੱਟਵਰਕ ਨਹੀਂ, ਕੋਈ ਅੱਪਲੋਡ ਨਹੀਂ, ਕੋਈ ਕਲਾਊਡ ਸਿੰਕ ਨਹੀਂ**।

- ਇੱਕ ਸਿੰਗਲ ਸੈਲਫ਼-ਕੰਟੇਨਡ HTML ਫਰੰਟਐਂਡ (ਕੋਈ Vue / React ਨਹੀਂ); Tauri v2 ਸਿਰਫ਼ ਨੇਟਿਵ ਸ਼ੈੱਲ ਦਿੰਦਾ ਹੈ (ਵਿੰਡੋਜ਼, ਮੇਨੂ, ਫਾਈਲ ਡਾਇਲਾਗ)।
- **ਜ਼ੀਰੋ ਰਨਟਾਈਮ ਨੈੱਟਵਰਕ ਬੇਨਤੀਆਂ**: `marked` / `KaTeX` / `highlight.js` / `DOMPurify` / `mermaid` / `jsPDF` / `html2canvas-pro` / `turndown` / `@retorquere/bibtex-parser` ਅਤੇ ਸਾਰੇ KaTeX woff2 ਫੌਂਟ ਇੱਕ ਸਿੰਗਲ `index.html` ਵਿੱਚ ਇਨਲਾਈਨ / base64-ਐਂਬੈੱਡ ਕੀਤੇ ਗਏ ਹਨ।
- `.md` / `.markdown` / `.html` ਸਪੋਰਟ ਕਰਦਾ ਹੈ; `.md` ਲਈ ਡਿਫ਼ਾਲਟ ਹੈਂਡਲਰ ਵਜੋਂ ਸੈੱਟ ਕੀਤਾ ਜਾ ਸਕਦਾ ਹੈ — ਡਬਲ-ਕਲਿੱਕ ਨਾਲ ਖੁੱਲ੍ਹਦਾ ਹੈ।

> ਢੁਕਵਾਂ: **ਖੋਜ / ਇੰਟ੍ਰਾਨੈੱਟ / ਪ੍ਰਾਈਵੇਸੀ ਸੁਰੱਖਿਆ** ਦ੍ਰਿਸ਼ਾਂ ਲਈ। ਕੋਈ ਇਸ਼ਤਿਹਾਰ ਨਹੀਂ, ਕੋਈ ਟੈਲੀਮੈਟਰੀ ਨਹੀਂ, ਕੋਈ ਡਾਟਾ ਅੱਪਲੋਡ ਨਹੀਂ।

## ✨ Features

- **ਮਲਟੀ-ਟੈਬ + ਮਲਟੀ-ਵਿੰਡੋ**: ਇੱਕ ਵਾਰ ਵਿੱਚ ਕਈ ਫਾਈਲਾਂ ਖੋਲ੍ਹੋ; ਨਾ-ਸੰਭਾਲੀਆਂ ਤਬਦੀਲੀਆਂ ਇੱਕ ਬਿੰਦੂ ਨਾਲ ਅੰਕਿਤ ਹੁੰਦੀਆਂ ਹਨ, ਅਤੇ ਬੰਦ ਕਰਨ ਤੋਂ ਪਹਿਲਾਂ ਤੁਹਾਨੂੰ ਪੁੱਛਿਆ ਜਾਂਦਾ ਹੈ; ਟੈਬ ਬੰਦ ਕਰਨ ਲਈ ਮਿਡਲ-ਕਲਿੱਕ ਕਰੋ। ਕਿਸੇ `.md` ਨੂੰ ਡਬਲ-ਕਲਿੱਕ ਕਰਨ ਨਾਲ ਉਸਦੀ ਆਪਣੀ ਵਿੰਡੋ ਖੁੱਲ੍ਹਦੀ ਹੈ (ਹਰ ਫਾਈਲ ਲਈ ਇੱਕ ਵਿੰਡੋ); ਪਹਿਲਾਂ ਤੋਂ ਖੁੱਲ੍ਹੀ ਫਾਈਲ ਨੂੰ ਡਬਲ-ਕਲਿੱਕ ਕਰਨ ਨਾਲ **ਉਸ ਵਿੰਡੋ 'ਤੇ ਫੋਕਸ** ਹੁੰਦਾ ਹੈ, ਮੁੜ-ਖੋਲ੍ਹਣ ਦੀ ਬਜਾਏ।
- **ਲਾਈਵ ਸਪਲਿਟ ਪ੍ਰੀਵਿਊ**: ਡਿਵਾਈਡਰ ਨੂੰ ਖਿੱਚੋ ਤੇ ਰੀਸਾਈਜ਼ ਕਰੋ; ਟੂਲਬਾਰ ਬਟਨ Split / Editor / Preview ਵਿੱਚ ਚੱਕਰ ਕਰਦਾ ਹੈ।
- **ਕਲਿੱਕ-ਟੂ-ਪੁਜ਼ੀਸ਼ਨ**: ਐਡੀਟਰ ਵਿੱਚ ਕਲਿੱਕ ਕਰੋ ਤੇ ਪ੍ਰੀਵਿਊ ਸਕ੍ਰੌਲ ਹੋਵੇ; ਪ੍ਰੀਵਿਊ ਵਿੱਚ ਕਲਿੱਕ ਕਰੋ ਤੇ ਐਡੀਟਰ ਵਿੱਚ ਕਰਸਰ ਜੰਪ ਕਰੇ।
- **ਖੋਜ ਤੇ ਬਦਲੋ**: ਲੱਭੋ, ਇੱਕ ਜਾਂ ਸਾਰੇ ਬਦਲੋ, ਮੈਚ ਗਿਣਤੀ ਸਹਿਤ।
- **ਗਣਿਤ**: ਇਨਲਾਈਨ `$…$` ਤੇ ਬਲਾਕ `$$…$$` (ਨਾਲ ਹੀ `\(...\)`, `\[...\]`), KaTeX ਵਲੋਂ ਰੈਂਡਰ; ਲੰਬੇ ਸਮੀਕਰਨ ਓਪਰੇਟਰਾਂ 'ਤੇ ਲਪੇਟਦੇ ਜਾਂ ਆਟੋ-ਸਿਕੋੜੇ ਹੁੰਦੇ ਹਨ।
- **ਕੋਡ ਹਾਈਲਾਈਟਿੰਗ**: ਭਾਸ਼ਾ ਆਟੋ-ਡਿਟੈਕਟ; ਵੱਡੇ ਡੌਕਸ ਵਿਊਪੋਰਟ ਮੁਤਾਬਕ ਲੇਜ਼ੀ-ਹਾਈਲਾਈਟ ਹੁੰਦੇ ਹਨ ਤਾਂ ਜੋ ਸਮੂਥ ਰਹੇ।
- **Mermaid ਡਾਇਆਗ੍ਰਾਮ**: ਇੱਕ ` ```mermaid ` ਬਲਾਕ ਫਲੋਚਾਰਟ / ਸੀਕੁਐਂਸ / ਕਲਾਸ / ਸਟੇਟ / ਗੈਨਟ / ਪਾਈ ਆਦਿ ਵਜੋਂ ਰੈਂਡਰ ਹੁੰਦਾ ਹੈ; ਡਾਇਆਗ੍ਰਾਮ 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਤਾਂ ਇੱਕ ਸਟੈਂਡਅਲੋਨ ਜ਼ੂਮ/ਪੈਨ/ਫੁੱਲਸਕ੍ਰੀਨ ਵਿਊਅਰ ਵਿੰਡੋ ਖੁੱਲ੍ਹਦੀ ਹੈ ਜੋ ਸੋਧਣ ਵੇਲੇ ਲਾਈਵ ਅੱਪਡੇਟ ਹੁੰਦੀ ਹੈ।
- **ਚਿੱਤ੍ਰ**: ਪੇਸਟ / ਡ੍ਰੌਪ / ਪਿਕ — ਦਸਤਾਵੇਜ਼ ਦੇ ਨਾਲ `<ਫਾਈਲਨਾਮ>_images/` ਫੋਲਡਰ ਵਿੱਚ ਸੰਭਾਲੇ ਜਾਂਦੇ ਹਨ, ਸਾਫ਼ ਸਾਪੇਖਿਕ ਹਵਾਲੇ ਨਾਲ (ਕੋਈ ਇਨਲਾਈਨ base64 ਨਹੀਂ); ਡਰਾਫਟ ਅਸਥਾਈ ਫੋਲਡਰ ਵਰਤਦੇ ਹਨ ਜੋ ਸੰਭਾਲਣ ਵੇਲੇ ਮਾਈਗ੍ਰੇਟ ਹੁੰਦਾ ਹੈ; «ਇੰਝ ਸੰਭਾਲੋ» ਚਿੱਤ੍ਰਾਂ ਨੂੰ ਨਿਸ਼ਾਨੇ ਵਿੱਚ ਫਲੈਟ ਕਰਦਾ ਹੈ; ਡਿਫ਼ਾਲਟ ਤੌਰ 'ਤੇ ਸੈਂਟਰ।
- **ਫੌਂਟ ਜ਼ੂਮ**: ਐਡੀਟਰ ਤੇ ਪ੍ਰੀਵਿਊ ਫੌਂਟ ਨੂੰ ਆਜ਼ਾਦ ਤੌਰ 'ਤੇ ਜ਼ੂਮ ਕਰੋ (−/ਪ੍ਰਤੀਸ਼ਤ/+ ਕੰਟਰੋਲ, ਜਾਂ `⌘/Ctrl + =/−/0`); ਮੁੜ-ਚਾਲੂ ਹੋਣ 'ਤੇ ਬਣਿਆ ਰਹਿੰਦਾ ਹੈ।
- **ਟੇਬਲ**: GFM ਟੇਬਲ; ਤੰਗ ਟੇਬਲ ਸਮੱਗਰੀ ਵੱਲ ਸੈਂਟਰ, ਚੌੜੇ ਕੱਟੇ ਬਿਨਾਂ ਹਰੀਜੌਂਟਲ ਸਕ੍ਰੌਲ ਹੁੰਦੇ ਹਨ।
- **ਹਵਾਲੇ (BibTeX)**: `[@key]` / `\cite{key}` ਸੰਟੈਕਸ, ਨਿਊਮੇਰਿਕ ਸਟਾਈਲ; ਅੰਤ ਵਿੱਚ ਹਵਾਲਾ ਸੂਚੀ ਤਿਆਰ ਹੁੰਦੀ ਹੈ, ਟੈਕਸਟ-ਅੰਦਰ `[n]` ਤੇ ਐਂਟਰੀ ਵਿਚਕਾਰ ਦੋ-ਪਾਸੀ ਜੰਪ ਹਨ; ਐਂਬੈੱਡਡ ` ```bibtex ` ਬਲਾਕ ਜਾਂ ਵੱਖਰੇ ਤੌਰ 'ਤੇ ਲੋਡ ਕੀਤੇ `.bib` ਨੂੰ ਸਪੋਰਟ ਕਰਦਾ ਹੈ।
- **HTML ਸਪੋਰਟ**: ਰੈਂਡਰ ਕਰਨ ਲਈ `.html` ਫਾਈਲਾਂ ਖੋਲ੍ਹੋ; HTML ਤੇ Markdown ਵਿਚਕਾਰ ਬਦਲੋ।
- **ਥੀਮ / ਭਾਸ਼ਾ**: ਡਾਰਕ / ਲਾਈਟ, **17 UI ਭਾਸ਼ਾਵਾਂ** (English, 简体中文, Français, Deutsch, Русский, Italiano, 日本語, 한국어, Español, Português, العربية, हिन्दी, ਪੰਜਾਬੀ, Tiếng Việt, Bahasa Indonesia, اردو, Монгол (Кирилл) — Arabic ਤੇ Urdu ਆਟੋ ਸੱਜੇ ਤੋਂ ਖੱਬੇ)।
- **ਆਟੋ-ਡਰਾਫਟ**: ਸਮੱਗਰੀ ਸਮੇਂ-ਸਮੇਂ 'ਤੇ ਸੰਭਾਲੀ ਜਾਂਦੀ ਹੈ ਤੇ ਅਚਾਨਕ ਬੰਦ / ਕ੍ਰੈਸ਼ ਮਗਰੋਂ ਰੀਸਟੋਰ ਹੁੰਦੀ ਹੈ।
- **ਸ਼ਬਦ ਗਿਣਤੀ**: ਸਟੇਟਸ ਬਾਰ ਅੱਖਰ / ਲਾਈਨਾਂ / ਸ਼ਬਦ ਲਾਈਵ ਵਿਖਾਉਂਦਾ ਹੈ, ਨਾਲ ਹੀ ਮੌਜੂਦਾ ਕਤਾਰ ਤੇ ਕਾਲਮ।
- **ਖਿੱਚੋ-ਤੇ-ਸੁੱਟੋ**: ਕੋਈ `.md` ਫਾਈਲ ਵਿੰਡੋ 'ਤੇ ਸੁੱਟੋ ਤੇ ਖੁੱਲ੍ਹ ਜਾਵੇ; ਚਿੱਤ੍ਰ ਸੁੱਟੋ ਤੇ ਜੁੜ ਜਾਵੇ।
- **ਐਕਸਪੋਰਟ**: Markdown / HTML / PDF (ਵੈਕਟਰ + ਰਾਸਟਰ) / LaTeX ਵਜੋਂ ਸੰਭਾਲੋ।

## 📝 Cheat sheet

**Markdown**: ਸਿਰਲੇਖ `# / ## / ###`, ਬੋਲਡ `**text**`, ਇਟੈਲਿਕ `*text*`, ਸਟ੍ਰਾਈਕਥ੍ਰੂ `~~text~~`, ਇਨਲਾਈਨ ਕੋਡ `` `code` ``, ਕੋਡ ਬਲਾਕ (ਤਿੰਨ backticks, ਵਿਕਲਪਿਕ ਭਾਸ਼ਾ ਸਹਿਤ), ਹਵਾਲਾ `> text`, ਸੂਚੀਆਂ `- / 1.`, ਟਾਸਕ ਸੂਚੀ `- [ ] / - [x]`, ਲਿੰਕ `[text](url)`, ਚਿੱਤ੍ਰ `![alt](url)`, ਡਿਵਾਈਡਰ `---`, ਟੇਬਲ `| A | B |`।

**ਗਣਿਤ**: ਇਨਲਾਈਨ `$E = mc^2$`; ਬਲਾਕ `$$\int_0^1 x\,dx$$` (ਕਈ ਲਾਈਨਾਂ ਵਿੱਚ ਫੈਲ ਸਕਦਾ ਹੈ)। LaTeX ਸੰਟੈਕਸ ਵਰਦਾ ਹੈ, KaTeX ਵਲੋਂ ਰੈਂਡਰ; ਕੋਡ ਬਲਾਕਾਂ ਅੰਦਰ `$` ਨੂੰ ਗਣਿਤ ਡੀਲੀਮੀਟਰ ਨਹੀਂ ਮੰਨਿਆ ਜਾਂਦਾ. `align` / `aligned`, ਮੈਟ੍ਰਿਕਸ, `cases` ਤੇ ਹੋਰ ਆਮ ਵਾਤਾਵਰਣ ਸਪੋਰਟ ਹਨ।

**ਹਵਾਲੇ**: ਟੈਕਸਟ ਵਿੱਚ `[@key]` ਜਾਂ `[@a; @b]` ਲਿਖੋ (LaTeX-ਅਨੁਕੂਲ `\cite{key}`), ਲਾਇਬ੍ਰੇਰੀ ਨੂੰ ਇੱਕ ` ```bibtex ` ਬਲਾਕ ਰਾਹੀਂ ਐਂਬੈੱਡ ਕਰੋ ਜਾਂ "Refs" ਬਟਨ ਨਾਲ ਕੋਈ `.bib` ਲੋਡ ਕਰੋ। ਅੰਤ ਵਿੱਚ ਹਵਾਲਾ ਸੂਚੀ ਤਿਆਰ ਹੁੰਦੀ ਹੈ; ਟੈਕਸਟ-ਅੰਦਰ `[n]` ਕਲਿੱਕ-ਯੋਗ ਹੈ।

## 📤 Export (Save As)

"Save As" 'ਤੇ ਕਲਿੱਕ ਕਰੋ ਤੇ ਫਾਰਮੈਟ ਚੁਣੋ:

- **Markdown (.md)**: ਸਰੋਤ ਸੰਭਾਲੋ ਤੇ ਮੌਜੂਦਾ ਟੈਬ ਦਾ ਨਾਮ / ਪਾਥ ਅੱਪਡੇਟ ਕਰੋ।
- **HTML (.html)**: ਸੈਲਫ਼-ਕੰਟੇਨਡ HTML, ਇਨਲਾਈਨ CSS + ਕੋਡ ਹਾਈਲਾਈਟਿੰਗ ਸਹਿਤ; ਗਣਿਤ `$…$` ਲਿਟਰਲ ਵਜੋਂ ਰੱਖਿਆ, ਇਨਲਾਈਨ KaTeX ਵਲੋਂ ਆਟੋ-ਰੈਂਡਰ।
- **PDF vector**: ਸਿਸਟਮ ਪ੍ਰਿੰਟ ਡਾਇਲਾਗ, ਵੈਕਟਰ ਆਉਟਪੁੱਟ, ਕਿਸੇ ਵੀ ਜ਼ੂਮ 'ਤੇ ਤਿੱਖਾ। "Save as PDF" ਚੁਣੋ।
- **PDF raster**: ਸਿੱਧਾ PDF ਫਾਈਲ ਤਿਆਰ ਕਰਦਾ ਹੈ (ਕੋਈ ਡਾਇਲਾਗ ਨਹੀਂ), ਘੱਟ / ਮੱਧਮ / ਉੱਚ ਰੈਜ਼ੋਲਿਊਸ਼ਨ ਸਹਿਤ; ਬਲਾਕ ਹੱਦਾਂ 'ਤੇ ਪੇਜ਼ਬੰਦੀ (ਕੋਈ ਟੁੱਟੇ ਸਮੀਕਰਨ / ਸਿਰਲੇਖ / ਕੋਡ ਨਹੀਂ)।
- **LaTeX (.tex)**: ਕੰਪਾਇਲ-ਯੋਗ `.tex` ਸਰੋਤ ਵਿੱਚ ਬਦਲਿਆ (documentclass ਤੇ packages ਸਹਿਤ; ਗਣਿਤ ਜਿਵੇਂ-ਦਾ-ਤਿਵੇਂ)। ਇੱਕ ਕਾਪੀ ਐਕਸਪੋਰਟ ਕਰਦਾ ਹੈ।

## 🔒 Offline & security

- **ਜ਼ੀਰੋ ਰਨਟਾਈਮ ਨੈੱਟਵਰਕ ਬੇਨਤੀਆਂ।** ਬਿਲਡ ਆਉਟਪੁੱਟ `dist/index.html` ਸੈਲਫ਼-ਚੈੱਕ ਹੁੰਦਾ ਹੈ: ਕੋਈ `src=` / `href=` / `url()` / `@import` ਬਾਹਰੀ ਲਿੰਕ ਨਹੀਂ।
- ਸਖ਼ਤ CSP (ਸਿਰਫ਼ ਲੋਕਲ IPC, ਕੋਈ WAN ਨਹੀਂ); ਸਾਰੀਆਂ ਫਾਈਲਾਂ ਲੋਕਲ ਤੌਰ 'ਤੇ ਪੜ੍ਹੀਆਂ/ਲਿਖੀਆਂ ਜਾਂਦੀਆਂ ਹਨ, ਕੁਝ ਵੀ ਅੱਪਲੋਡ ਨਹੀਂ।
- ਜਾਂਚੋ: Wi-Fi ਬੰਦ ਕਰੋ / ਕੇਬਲ ਕੱਢੋ ਤੇ ਲਾਂਚ ਕਰੋ — ਗਣਿਤ, ਚਿੱਤ੍ਰ, ਕੋਡ ਹਾਈਲਾਈਟਿੰਗ ਤੇ Mermaid ਸਭ ਕੰਮ ਕਰਦੇ ਹਨ।
- `dist/index.html` ਵਿੱਚ ਅਜੇ ਵੀ ~ਇੱਕ ਦਰਜਨ `https://github.com/…` ਸਤਰਾਂ ਦਿਖਾਈ ਦਿੰਦੀਆਂ ਹਨ; ਇਹ ਸਭ `marked` / `highlight.js` ਆਦਿ ਦੇ **ਲਾਇਸੈਂਸ / ਸਰੋਤ ਟਿੱਪਣੀਆਂ** ਅੰਦਰ ਹਨ — ਸਾਦਾ ਟੈਕਸਟ ਜੋ **ਕਦੇ ਬੇਨਤੀ ਨਹੀਂ ਚਲਾਉਂਦਾ**; ਓਪਨ-ਸੋਰਸ ਲਾਇਸੈਂਸਾਂ ਦਾ ਆਦਰ ਕਰਨ ਲਈ ਜਿਵੇਂ-ਦਾ-ਤਿਵੇਂ ਛੱਡਿਆ।

### Opening the unsigned app on macOS (bypass Gatekeeper)

ਇਹ ਐਪ ਡਿਵੈਲਪਰ-ਸਾਈਨ / ਨੋਟਰਾਈਜ਼ਡ **ਨਹੀਂ** ਹੈ (ਆਫਲਾਈਨ ਦ੍ਰਿਸ਼ ਆਮ ਤੌਰ 'ਤੇ ਆਨਲਾਈਨ ਨੋਟਰਾਈਜ਼ ਨਹੀਂ ਕਰ ਸਕਦੇ)। macOS 12+ 'ਤੇ, **ਖਾਸ ਕਰਕੇ macOS 26 (Tahoe)**, ਇਸਨੂੰ ਸਿੱਧਾ `.dmg` ਤੋਂ — ਜਾਂ ਤਾਜ਼ੀ ਕਾਪੀ ਕੀਤੇ ਬਿਲਡ ਤੋਂ — ਲਾਂਚ ਕਰਨ 'ਤੇ **"MDeX.app is damaged and can't be opened."** ਨਾਲ ਫੇਲ੍ਹ ਹੁੰਦਾ ਹੈ। ਇਹ Gatekeeper ਹੈ, ਅਸਲ ਨੁਕਸਾਨ ਨਹੀਂ। Terminal ਵਿੱਚ ਠੀਕ ਕਰੋ:

1. **ਪਹਿਲਾਂ `MDeX.app` ਨੂੰ `.dmg` ਤੋਂ ਬਾਹਰ ਕੱਢ ਕੇ `/Applications` ਵਿੱਚ ਖਿੱਚੋ** — ਇਸਨੂੰ dmg ਤੋਂ ਸਿੱਧਾ ਕਦੇ ਨਾ ਚਲਾਓ (ਇਹ App Translocation ਤੇ `com.apple.provenance` ਐਟ੍ਰੀਬਿਊਟ ਨੂੰ ਚਾਲੂ ਕਰਦਾ ਹੈ, macOS 26 'ਤੇ "damaged" ਦਾ ਅਸਲ ਕਾਰਨ)।
2. ਐਟ੍ਰੀਬਿਊਟ ਸਾਫ਼ ਕਰੋ ਤੇ ਮੁੜ-ਸਾਈਨ:
   ```bash
   xattr -cr /Applications/MDeX.app
   codesign --force --deep --sign - /Applications/MDeX.app
   ```
   > `com.apple.provenance` SIP-ਸੁਰੱਖਿਅਤ ਹੈ ਤੇ `sudo` ਨਾਲ ਵੀ **ਹਟਾਇਆ ਨਹੀਂ** ਜਾ ਸਕਦਾ; ਮੁੜ-ਸਾਈਨਿੰਗ ਦਸਤਖ਼ਤ ਰੀਸੈਟ ਕਰਦੀ ਹੈ ਤਾਂ ਜੋ Gatekeeper ਇਸਨੂੰ ਚੱਲਣ ਦੇਵੇ। `spctl` ਅਜੇ ਵੀ ad-hoc ਸਾਈਨਿੰਗ ਲਈ `rejected` ਦੱਸਦਾ ਹੈ — ਉਮੀਦ-ਮੁਤਾਬਕ, ਤੇ ਇਹ `open` ਨੂੰ **ਰੋਕਦਾ ਨਹੀਂ**।
3. `open /Applications/MDeX.app` ਨਾਲ ਲਾਂਚ ਕਰੋ (ਜਾਂ ਡਬਲ-ਕਲਿੱਕ)। ਪਹਿਲੀ ਲਾਂਚ ਇੱਕ ਵਾਰ ਪੁੱਛ ਸਕਦੀ ਹੈ — **System Settings → Privacy & Security → Open Anyway** ਰਾਹੀਂ ਤਸਦੀਕ ਕਰੋ, ਜਾਂ ਐਪ 'ਤੇ ਸੱਜਾ-ਕਲਿੱਕ → **Open**।

## 📁 Project structure

```
markdown/
├── app-shell.html          # frontend source (HTML+CSS+JS, all app logic)
├── tools/
│   ├── fetch-vendor.mjs    # one-time: download deps into vendor/ (online only here)
│   └── build-html.mjs      # inline vendor into dist/index.html (KaTeX fonts → base64)
├── dist/index.html         # build output: fully offline single file (Tauri frontendDist)
├── vendor/                 # download cache (.gitignore)
├── package.json            # @tauri-apps/cli + scripts
└── src-tauri/
    ├── Cargo.toml          # tauri 2 + tauri-plugin-dialog / single-instance
    ├── build.rs            # tauri_build::build()
    ├── tauri.conf.json     # 1200×750 window, strict CSP, icons, .md association, menu hooks
    ├── capabilities/default.json
    ├── icons/              # full icon set from `cargo tauri icon`
    └── src/{main.rs, lib.rs}   # menus + file IO + multi-window routing
```
