# MDeX v1.3.3 (macOS · Windows · Linux · مکمل آف لائن · Tauri v2)

> **MDeX** · تلفظ "em-dex" (/ˌemˈdɛks/) — حرف M کے بعد "dex"، دو حصے۔

ایک آف لائن اول Markdown ریڈر اور ایڈیٹر، **air-gapped / intranet / disconnected** استعمال کے لیے۔ ہر فائل مقامی طور پر پروسیس ہوتی ہے — **کوئی نیٹ ورک نہیں، کوئی اپ لوڈ نہیں، کوئی کلاؤڈ سنک، بغیر اشتہارات، بغیر ٹیلی میٹری، بغیر ڈیٹا اپ لوڈ نہیں**۔

- ایک واحد خود کفیل HTML فرنٹ اینڈ (کوئی Vue / React نہیں)؛ Tauri v2 صرف نیٹو شیلی فراہم کرتا ہے (ونڈوز، مینیوز، فائل ڈائیلاگز)۔
- **زیرو رن ٹائم نیٹ ورک درخواستیں**: `marked` / `KaTeX` / `highlight.js` / `DOMPurify` / `mermaid` / `jsPDF` / `html2canvas-pro` / `turndown` / `@retorquere/bibtex-parser` اور تمام KaTeX woff2 فونٹس ایک واحد `index.html` میں اِن لائن / base64-ایمبیڈڈ ہیں۔
- `.md` / `.markdown` / `.html` کی سپورٹ؛ بطور ڈیفالٹ `.md` ہینڈلر سیٹ کیا جا سکتا ہے — کھولنے کے لیے ڈبل کلک کریں۔

> ان کے لیے موزوں ہے: **تحقیق / انٹرانیٹس / رازداری کا تحفظ** کے منظرنامے۔ کوئی اشتہارات نہیں، کوئی ٹیلی میٹری نہیں، کوئی ڈیٹا اپ لوڈ نہیں۔

## 📝 چیٹ شیٹ

**Markdown**: ہیڈنگز `# / ## / ###`، بولڈ `**text**`، اٹیلک `*text*`، اسٹرائیک تھرو `~~text~~`، اِن لائن کوڈ `` `code` ``، کوڈ بلاکس (تگنا بیک ٹکس، اختیاری زبان کے ساتھ)، کوٹ `> text`، لسٹس `- / 1.`، ٹاسک لسٹ `- [ ] / - [x]`، لنک `[text](url)`، تصویر `![alt](url)`، ڈیوائیڈر `---`، ٹیبلز `| A | B |`۔

**ریاضی**: اِن لائن `$E = mc^2$`؛ بلاک `$$\int_0^1 x\,dx$$` (متعدد لائنز پر محیط ہو سکتا ہے)۔ LaTeX نحو استعمال کرتا ہے، KaTeX کے ذریعے رینڈر ہوتا ہے؛ کوڈ بلاکس کے اندر `$` کو ریاضی کے ڈیلمیٹر کے طور پر تسلیم نہیں کیا جاتا۔ `align` / `aligned`، میٹریسز، `cases` اور دیگر عام ماحولیات کی سپورٹ۔

**حوالہ جات**: متن میں `[@key]` یا `[@a; @b]` لکھیں (LaTeX-مطابق `\cite{key}`)، لائبریری کو ` ```bibtex ` بلاک کے ذریعے ایمبیڈ کریں یا "Refs" بٹن کے ذریعے `.bib` لوڈ کریں۔ آخر میں References فہرست بنائی جاتی ہے؛ اِن ٹیکسٹ `[n]` کلک کے قابل ہے۔

## 📤 ایکسپورٹ (بطور محفوظ کریں)

"Save As" پر کلک کریں اور ایک فارمیٹ منتخب کریں:

- **Markdown (.md)**: سورس محفوظ کریں اور موجودہ ٹیب کا نام / پاتھ اپ ڈیٹ کریں۔
- **HTML (.html)**: خود کفیل HTML، اِن لائن CSS + کوڈ ہائی لائٹنگ کے ساتھ؛ ریاضی بطور `$…$` لٹرل رکھی گئی، اِن لائن KaTeX کے ذریعے خود کار طور پر رینڈر ہوتی ہے۔

- **LaTeX (.tex)**: ایک کمپائل کے قابل `.tex` سورس میں تبدیل (documentclass اور پیکجز کے ساتھ؛ ریاضی جیسا ہے ویسا ہی رکھی گئی)۔ ایک کاپی ایکسپورٹ کرتا ہے۔

### macOS پر غیر دستخط شدہ ایپ کھولنا (Gatekeeper کو بائی پاس کرنا)

یہ ایپ ڈویلپر-دستخط شدہ / نوٹرائزڈ **نہیں** ہے (آف لائن منظرنامے عام طور پر آن لائن نوٹرائز نہیں کر سکتے)۔ macOS 12+ پر، **خاص طور پر macOS 26 (Tahoe)**، اسے سیدھے `.dmg` سے — یا تازہ کاپی شدہ بلڈ سے — لانچ کرنا **"MDeX.app is damaged and can't be opened."** کے ساتھ ناکام ہوتا ہے۔ یہ Gatekeeper ہے، حقیقی نقصان نہیں۔ ٹرمنل میں درست کریں:

1. **پہلے `MDeX.app` کو `.dmg` سے نکال کر `/Applications` میں ڈالیں** — اسے dmg سے براہ راست کبھی نہ چلائیں (یہ App Translocation اور `com.apple.provenance` ایٹریبیوٹ کو متحرک کرتا ہے، جو macOS 26 پر "damaged" کا اصل سبب ہے)۔
2. ایٹریبیوٹس صاف کریں اور دوبارہ دستخط کریں:
   ```bash
   xattr -cr /Applications/MDeX.app
   codesign --force --deep --sign - /Applications/MDeX.app
   ```
   > `com.apple.provenance` SIP کے ذریعے محفوظ ہے اور `sudo` کے ساتھ بھی **ہٹائی** نہیں جا سکتی؛ دوبارہ دستخط کرنے سے دستخط ری سیٹ ہو جاتے ہیں تاکہ Gatekeeper اسے چلانے دے۔ `spctl` اب بھی ایڈ ہاک دستخط کے لیے `rejected` رپورٹ کرتا ہے — یہ متوقع ہے، اور یہ `open` کو **روک** نہیں کرتا۔
3. `open /Applications/MDeX.app` کے ساتھ لانچ کریں (یا ڈبل کلک کریں)۔ پہلا لانچ اب بھی ایک بار اشارہ کر سکتا ہے — **System Settings → Privacy & Security → Open Anyway** کے ذریعے تصدیق کریں، یا ایپ پر دائیں کلک کریں → **Open**۔

│   └── build-html.mjs      # inline vendor into dist/index.html (KaTeX fonts → base64)

    ├── Cargo.toml          # tauri 2 + tauri-plugin-dialog / single-instance
    ├── build.rs            # tauri_build::build()
    ├── tauri.conf.json     # 1200×750 window, strict CSP, icons, .md association, menu hooks
    ├── capabilities/default.json
    ├── icons/              # full icon set from `cargo tauri icon`

## 🎨 کسٹمائزیشن

| تبدیل کرنے کے لیے | کہاں |
| --- | --- |
| ایپ کا نام / Bundle ID | `src-tauri/tauri.conf.json` → `productName` / `identifier` |
| ونڈو کا سائز | `tauri.conf.json` → `app.windows[0]` (ڈیفالٹ 1200×750) |
| آئیکنز | سورس تصویر تبدیل کریں، پھر `npm run icon` |
| تھیم رنگز / فونٹس | `app-shell.html` کے شروع میں `:root` CSS ویری ایبلز |
| مینیو آئٹمز | `src-tauri/src/lib.rs` میں `build_menu()` |
| UI سٹرنگز / ہیلپ دستاویز | `app-shell.html` میں `I18N` / `HELP_STRINGS` |
| ڈیپنڈنسی ورژنز | `tools/fetch-vendor.mjs` کے شروع میں `VERSIONS` (پھر `npm run fetch -- --force`) |
