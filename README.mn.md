# MDeX v1.2.0 (macOS · Windows · Linux · Бүрэн офлайн · Tauri v2)

> **MDeX** · "em-dex" (/ˌemˈdɛks/) гэж дуудна — M үсгийн дараа "dex" гэж бичигдсэн, хоёр үе бүхий үг.

**Агаараар тусгаарлагдсан / интранет / холбоогүй** орчинд зориулсан офлайн тэргүүлэгч Markdown уншигч ба засварлагч. Файл бүр орон нутагдаа боловсрууллагддаг — **сүлжээ байхгүй, байршуулалт байхгүй, үүлэн синк байхгүй**.

- Нэг бие даасан HTML фронтэнд (Vue / React байхгүй); Tauri v2 зөвхөн үндсэн биерүүг (цонх, цэс, файлын диалог) хангана.
- **Ажиллах үеийн сүлжээний хүсэлт байхгүй**: `marked` / `KaTeX` / `highlight.js` / `DOMPurify` / `mermaid` / `jsPDF` / `html2canvas-pro` / `turndown` / `@retorquere/bibtex-parser` болон бүх KaTeX woff2 фонт нь нэг `index.html` дотор шингээгдсэн / base64-д суулгасан.
- `.md` / `.markdown` / `.html` дэмждэг; үндсэн `.md` нээгчээр тохируулж болно — давхар дарж нээдэг.

> Зохистой: **судалгаа / интранет / нууцлалын хамгаалалт**. Реклам байхгүй, телеметри байхгүй, өгөгдөл байршуулдаггүй.

## ✨ Боломжууд

- **Олон таб + олон цонх**: нэгэн зэрэг хэд хэдэн файл нээх; хадгалагдаагүй өөрчлөлтүүд цэгээр тэмдэглэгдэж, хаахын өмнө асууна; табыг хаахын тулд дунд дарна. `.md`-г давхар дарвал өөрийн гэсэн цонх нээгдэнэ (файл бүрд нэг цонх); аль нээлттэй файлыг давхар дарвал дахин нээхийн оронд **тэр цонх руу фокус хийнэ**.
- **Амьдралтай хуваагдсан урьдчилан харах**: хуваагчийг чирж хэмжээг өөрчилнө; хэрэгслийн мөрний товч Хуваагдсан / Засварлагч / Урьдчилан харах гэж ээлжилнэ.

- **Хайж бүхэлд нь сольих**: олж, нэгийг буюу бүгдийг сольих, тааралдсан тоог харуулна.
- **Математик**: шугаман `$…$` ба блок `$$…$$` (бас `\(...\)`, `\[...\]`), KaTeX-ээр дүрслэгдэнэ; урт тэгшитгэл оператор дээр мөрлүүд эсвэл автоматаар жижигрэнэ.

## 📝 Товч мэдээллийн хуудас

**Markdown**: гарчиг `# / ## / ###`, бүдүүн `**text**`, налуу `*text*`, дундуураа зураастай `~~text~~`, шугаман код `` `code` ``, кодын блокууд (гурван backtick, хэлний сонголттой эсвэл үгүй), ишлэл `> text`, жагсаалт `- / 1.`, даалгаврын жагсаалт `- [ ] / - [x]`, холбоос `[text](url)`, зураг `![alt](url)`, хуваагч `---`, хүснэгт `| A | B |`.

**Математик**: шугаман `$E = mc^2$`; блок `$$\int_0^1 x\,dx$$` (мөрөнд хамаарч болно). LaTeX бичигдэх хэв маяг ашигладаг, KaTeX-ээр дүрслэгдэнэ; кодын блок доторх `$` нь математик тусгаарлагч байдлаар үзэхгүй. `align` / `aligned`, матриц, `cases` болон бусад нийтлэг орчнуудыг дэмждэг.

**Эшлэлүүд**: бичвэрт `[@key]` эсвэл `[@a; @b]` бичнэ (LaTeX-д нийцтэй `\cite{key}`), номын санг ` ```bibtex ` блокоор шингээх эсвэл "Refs" товчоор `.bib` ачаална. Төгсгөлд нь References жагсаалт бий болно; бичвэр доторх `[n]` дээр дарж болно.

## 📤 Экспорт (Өөрөөр хадгалах)

"Өөрөөр хадгалах"-г дарж форматаа сонгоно уу:

- **Markdown (.md)**: эхийг хадгалах ба одоогийн табын нэр / замыг шинэчлэх.
- **HTML (.html)**: биеэ даасан HTML ба CSS + кодын өнгө оруулалт шингээгдсэн; математик нь `$…$` бичвэрээр хадгалагдаж, шингээсэн KaTeX-ээр автоматаар дүрслэгдэнэ.
- **PDF вектор**: системийн хэвлэх диалог, вектор гаралт, ямар ч хэмжээсэнд тод. "Save as PDF"-г сонгоно уу.
- **PDF растер**: PDF файлыг шууд үүсгэх (диалог байхгүй), бага / дунд / өндөр нарийвчлалтай; блокийн хил дээр хуудасладаг (тэгшитгэл / гарчиг / код гэмтэхгүй).
- **LaTeX (.tex)**: эмхэтгэгдэх боломжтой `.tex` эх рүү хөрвүүлэгдэнэ (documentclass болон багцуудтай; математик хэвээрээ). Нэг хувь экспортлоно.

### macOS дээр гарын үсэггүй аппыг нээх (Gatekeeper-ийг тойрох)

Энэ апп нь хөгжүүлэгчийн гарын үсэггүй / notarize хийгдээгүй (офлайн тохиолдолд ихэвчлэн онлайнаар notarize хийх боломжгүй). macOS 12+, **түүний дотор macOS 26 (Tahoe)** дээр `.dmg`-ээс шууд эсвэл шинээр хуулсан бүтээгдэхүүнийг ажиллуулах нь **"MDeX.app is damaged and can't be opened."** гэсэн алдаатай бүтэлгүйтэнэ. Энэ бол Gatekeeper, бодит гэмтэл биш. Терминалд засна:

1. **`MDeX.app`-г эхлээд `.dmg`-ээс `/Applications` рүү чирж аваарай** — dmg-ээс шууд бүү ажиллуул (энэ нь App Translocation болон `com.apple.provenance` шинж чанарыг өдөөж, macOS 26 дээрх "damaged"-ийн бодит шалтгаан болно).
2. Шинж чанарыг арилгаж дахин гарын үсэг зурна:
   ```bash
   xattr -cr /Applications/MDeX.app
   codesign --force --deep --sign - /Applications/MDeX.app
   ```
   > `com.apple.provenance` нь SIP-аар хамгаалагдсан бөгөөд `sudo`-тай ч арилгах **боломжгүй**; дахин гарын үсэг зурснаар гарын үсгийг шинэчилж Gatekeeper ажиллахыг зөвшөөрнө. `spctl` нь ad-hoc гарын үсэгт `rejected` гэж хэвээр мэдэгдэнэ — ингэх ёстой, ба `open`-ийг ** саатуулдаггүй**.
3. `open /Applications/MDeX.app`-аар ажиллуулна (эсвэл давхар дарна). Анхны ажиллагаа нь нэг удаа асууж магадгүй — **System Settings → Privacy & Security → Open Anyway**-аар баталгаажуулна, эсвэл апп дээр баруун дарж → **Open**.
